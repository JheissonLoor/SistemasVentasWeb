# Roadmap

## Version Actual

- Proyecto universitario Java Web clasico.
- Login con roles.
- CRUD de empleados, clientes y productos.
- Registro, listado, filtros y anulacion de ventas.
- Reporte basico de productos mas vendidos.
- Maven y GitHub Actions.

## Proximas Mejoras

- Agregar pruebas unitarias para DAOs y utilidades.
- Agregar pruebas de integracion con una base temporal.
- Agregar paginacion y busqueda en tablas grandes.
- Agregar detalle expandido de cada venta.
- Agregar dashboard mensual con ventas por dia.
- Agregar control de caja y arqueo.
- Mejorar auditoria de acciones: usuario, fecha y accion.
- Migrar a Spring Boot, Spring Security y JPA.
- Separar frontend y backend si el proyecto evoluciona a version profesional.
